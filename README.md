# 适配hyperf的聚合美的支付平台SDK
## Installing

```shell
$ composer require nahuomall/mideapay -vvv
```

## Usage

```php
        $obj = \Hyperf\Support\make(\Midea\Api\Wechat::class)
        
        $obj->setMchId('你的商户ID');
        // app
        $res =$obj->app->createOrder([]);
        
        var_dump($res);
```
